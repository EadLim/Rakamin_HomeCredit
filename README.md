# Rakamin_HomeCredit
Repository ini berisi kode dan notebook untuk membangun model prediksi gagal bayar nasabah Home Credit, termasuk preprocessing, training, evaluasi metrik (AUC, precision–recall), serta visualisasi insight utama untuk presentasi bisnis.

Judul project: “Home Credit Default Risk – Credit Scoring Model”.

Deskripsi singkat: 
Membangun model klasifikasi untuk memprediksi probabilitas gagal bayar.
Memberikan rekomendasi kebijakan kredit & kampanye marketing berdasarkan insight data.

Cara menjalankan:

Environment (Python 3.x, install package via pip install -r requirements.txt).

Ringkasan hasil:

Model yang dipakai (LogReg & Random Forest) dan metrik utama (AUC, recall default).
